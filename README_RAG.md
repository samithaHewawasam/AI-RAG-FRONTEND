# 🚀 RAG Chat - Real-time RAG Assistant

A beautiful React chat interface with real-time visualization of the RAG (Retrieval-Augmented Generation) pipeline.

## ✨ Features

- 💬 **Floating Chat Widget** - Minimalist floating button that expands to full chat
- 🎨 **Beautiful UI** - Modern gradient design with smooth animations  
- 👤 **Smart Bubbles** - User messages on right (green), bot on left (purple)
- ⚙️ **Live Process View** - Collapsible pipeline visualization
- 📊 **Real-time Updates** - Server-Sent Events (SSE) streaming
- 📱 **Responsive** - Works on desktop and mobile
- ⌨️ **Keyboard Support** - Press Enter to send
- 🎯 **Type Safety** - Built with TypeScript

## 🚀 Quick Start

### 1. Start Backend
```bash
cd /Users/samithahewawasam/Documents/workspace/kodez/AI-solutions/ollama-rag
npm run start:dev
```

### 2. Start React App
```bash
cd rag-chat
npm start
```

### 3. Use the Chat
1. Click floating chat button (💬)
2. Type your question
3. Click "📊 Show Process" to see RAG pipeline
4. Watch real-time updates!
